# App Dependencies

The dependencies required to run the project :

- `Node`
- `Angular`
- `npm`
- `AWS CLI`
- `EB CLI`
- `cors`
- `dotenv`
- `express`
- `sequelize`
- `RDS` database running Postgres.
- `S3` bucket for hosting uploaded pictures.
- `Elastic Beanstalk` hosting API server
- `Zone.js`
- `bcrypt`
- `body-parser`
- `jsonwebtoken`
- `email-validator`
- `pg`
- `reflect-metadata`
